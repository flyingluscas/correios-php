Uma maneira fácil de usar o cálculo de frete dos [Correios](http://www.correios.com.br) para vários itens.

## Instalação

Via Composer

``` bash
$ composer require flyingluscas/correios-php
```

## Uso

``` php
use FlyingLuscas\Correios\Service;
use FlyingLuscas\Correios\Correios;

Correios::freight()
    ->origin('01001-000')
    ->destination('87047-230')
    ->services(Service::SEDEX, Service::PAC)
    ->items([
        ['width' => 16, 'height' => 16, 'length' => 16, 'weight' => 0.3, 'quantity' => 1],
        ['width' => 16, 'height' => 16, 'length' => 16, 'weight' => 0.3, 'quantity' => 3],
    ])
    ->calculate();
```
